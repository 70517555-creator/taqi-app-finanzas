# TAQI - App de Finanzas

## Descripción
Aplicación para gestionar ingresos y gastos personales.

## Problema
Mala administración del dinero en las personas.

## Solución
TAQI permite registrar ingresos, gastos y calcular el saldo disponible, promoviendo el ahorro.

## Tecnologías
- Python
- Tkinter

- docs/: documentación
